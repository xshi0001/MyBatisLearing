2.2.3. 传递pojo包装对象
	开发中通过可以使用pojo传递查询条件。
查询条件可能是综合的查询条件，不仅包括用户查询条件还包括其它的查询条件（比如查询用户信息的时候，将用户购买商品信息也作为查询条件），这时可以使用包装对象传递输入参数。
包装对象：Pojo类中的一个属性是另外一个pojo。

需求：查询订单表order的所有数据

需求：根据性别和名字查询用户

需求：根据多个id查询用户信息
